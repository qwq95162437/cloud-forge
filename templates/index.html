完成流程

从购买服务器到项目上线，形成完整的实战闭环。

01

购买与配置 ECS

选择实例、镜像、磁盘、安全组与公网访问方式。

02

系统初始化

登录 Linux，更新软件源，安装 Nginx、Git、Python 等环境。

03

开发静态网页

使用 VS Code 编写 HTML、CSS、JS，完成页面布局与交互。

04

部署 Web 服务

上传项目到 ECS，配置 Nginx，发布静态网页。

05

扩展动态功能

接入 Flask、Gunicorn 与 MySQL，形成动态网站基础能力。

06

运行与维护

日志监控、服务重启、脚本自动化与 Docker 容器运维。
